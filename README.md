# HomeAI Backend Node.js Project

## Project Structure

- `src/` - Source code (entry: `index.js`)
- `test/` - Test files
- `.gitignore` - Node and environment ignores
- `eslint.config.mjs` - ESLint configuration

## Getting Started

1. **Install dependencies:**
   ```sh
   npm install
   ```
2. **Run the project:**
   ```sh
   node src/index.js
   ```
3. **Lint the code:**
   ```sh
   npx eslint src/
   ```

## Customization
- Add your code in `src/`.
- Add tests in `test/`.
- Update ESLint config as needed.
